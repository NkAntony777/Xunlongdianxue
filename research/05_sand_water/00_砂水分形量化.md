### 2.3 水口

- **水口 = 水流出口处的两侧山**
- **天门**：水流入口，宜开阔
- **地户**：水流出口，宜封闭
- **关锁**：水口两侧山要"交牙"、有山形障碍

**算法化**：

```python
def score_water_outlet(river, water_outlet_point, dem, search_radius=1000):
    """水口评分：两侧山势是否关锁。"""

    outlet = water_outlet_point
    # 2. 沿水流方向两侧搜索显著高点
    left_peak = find_nearest_peak(dem, outlet, bearing=270, max_dist=search_radius)
    right_peak = find_nearest_peak(dem, outlet, bearing=90, max_dist=search_radius)
    # 3. 评分：两侧山高且与水口距离 < 200m
    if left_peak and right_peak:
        d_left = distance(outlet, left_peak)
        d_right = distance(outlet, right_peak)
        h_left = left_peak.elevation - outlet.elevation
        h_right = right_peak.elevation - outlet.elevation
        # 距离适中 + 高度可观
        if 50 < d_left < 500 and 50 < d_right < 500:
            return 100 * (1 - abs(d_left - d_right) / max(d_left, d_right))
    return 50
```

```python
    dy = np.gradient(ys)

    normal = np.array([-tangent[1], tangent[0]])
    # 法线是否指向候选穴
    dot = np.dot(vec_to_cand, normal)
    if dot > 0:
        return "玉带水（环抱）"
    else:
        return "反弓水"
```
