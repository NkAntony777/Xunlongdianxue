# 寻龙点穴引擎 - 速查卡

> 一页纸速查：所有关键算法 + 参数 + 公式

---

## 0. 流程图

```
DEM + Water/Road GeoJSON
        ↓
[1. 预处理]
  - Pit Fill
  - D8 Flow Direction
  - Flow Accumulation
  - Slope / Aspect / Hillshade
        ↓
[2. 龙脉识别]
  - 汇流累积=0 → 山脊线栅格
  - 骨架化 → 山脊线 vector
  - 简化 + 分级（Strahler-like）
  - 末端节点 → 入首点
        ↓
[3. 候选穴生成]
  - 入首点周围 100-500m 滑动窗口
  - 计算 TPI / TWI / 形态
  - 非极大值抑制（≥ 200m 间隔）
        ↓
[4. 砂水分析]
  - 候选穴周围水系距离 / 方向
  - 候选穴周围砂山 / 案山 / 朝山
  - 玉带水 / 反弓水判定
        ↓
[5. 四象评分]
  - 扇区统计（4 个 90° 扇区）
  - 龙虎 / 朱雀 / 玄武 评分
        ↓
[6. 综合评分]
  - 加权融合 → 0-100
  - 排序 → Top-K 候选
        ↓
[7. 输出]
  - JSON 报告
  - GeoJSON 候选穴
  - 评分热力栅格
  - LLM 解读（可选）
```

| 阈值 | 数值 | 来源 |
|------|------|------|

| **靠山距候选穴** | 50-500 m | 传统 |
| **案山距候选穴** | 200-1000 m | 传统 |
| **朝山距候选穴** | 1-5 km | 传统 |
| **水系最佳距离** | 50-1000 m | 参考开源 |
| **山脊线蜿蜒度** | ≥ 1.3 吉 | 真龙 |
| **山脊线分形维数** | ≥ 1.3 吉 | 青龙动 |
| **剥换次数** | ≥ 3 贵 | 传统 |
| **TPI 窝穴阈值** | < -0.5 | 局部凹 |
| **TPI 突穴阈值** | > 1.0 | 局部凸 |
| **TWI 高分位** | > 8 | 强聚气 |
| **Concave Hull ratio** | < 0.7 | 强凹形 |

## 2. 核心公式速查

### 2.1 坡度（Slope）

```python
slope = np.degrees(np.arctan(np.sqrt(dz/dx)**2 + (dz/dy)**2))
```

### 2.2 坡向（Aspect）

```python
aspect = (np.degrees(np.arctan2(-dz/dx, dz/dy)) + 360) % 360
```

### 2.3 TPI（Topographic Position Index）

```python
TPI = z_0 - mean(z_in_window)
```

### 2.4 TWI（Topographic Wetness Index）

```python
TWI = ln(a / tan(slope_rad))
# a: 上游汇流累积面积（栅格数 × 像元面积）
```

### 2.5 山脊线 = 汇流累积 = 0

```python
ridge_mask = (flow_accumulation == 0) & np.isfinite(dem)
```

### 2.6 Concave Hull

```python
from shapely import concave_hull
hull = concave_hull(multipoint, ratio=0.3)  # 0-1
```

### 2.7 综合评分

```python
overall = (
    0.20 * back_mountain_score   # 玄武
  + 0.20 * qinglong_baihu_score  # 龙虎
  + 0.18 * zhuque_score          # 朱雀
  + 0.15 * form_score            # 形态
  + 0.12 * water_score           # 水
  + 0.15 * stability_score       # 稳定性
)
```

### 2.8 24 山向

```python
def degree_to_24_mountain(deg):
    mountains = ['壬','子','癸','丑','艮','寅','甲','卯','乙','辰',
                 '巽','巳','丙','午','丁','未','坤','申','庚','酉',
                 '辛','戌','乾','亥']
    return mountains[int(((deg + 7.5) % 360) / 15)]
```

### 2.9 8 方位

```python
def degree_to_8_aspect(deg):
    names = ['北','东北','东','东南','南','西南','西','西北']
    return names[int(((deg + 22.5) % 360) / 45)]
```

|---------|------|------|
