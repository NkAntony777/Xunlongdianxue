## 4. 坐标约定（强制）

```text
facing  = 朝向（面朝），北=0，东=90，南=180，西=270
sit     = facing + 180          # 玄武
青龙    = facing + 270          # 左
白虎    = facing + 90           # 右
朱雀    = facing                # 前
玄武    = sit                   # 后
```

默认坐北朝南：`facing=180` → 青龙东、白虎西、玄武北、朱雀南。
