## 1. 纲领：《葬经》与「风水」二字

通行本《葬经》（托名郭璞，思想源魏晋至唐宋）给出堪舆元语言：

| 原文要旨 | 白话 | 工程含义 |
|----------|------|----------|
| 葬者，乘生气也 | 择地核心是承接「地中流行之气」 | 目标函数 = 生气可聚可止 |
| 气乘风则散，界水则止 | 风散气，水止气 | **藏风** + **界水** 是几何约束 |
| **风水之法，得水为上，藏风次之** | 得水优先，藏风其次 | **得水 ≠ 无煞**；得水是吉项，另有水煞凶项 |
| 势来形止，全气之地，当葬其止 | 远势来、近形止处结穴 | 先龙脉（势）后微地形（形） |
| 千尺为势，百尺为形 | 势大形小 | 多尺度：km 级龙 / 百米级穴 |

民间口诀与操作顺序：

```text
入山寻水口 → 寻龙看祖宗 → 入首看剥换 → 登穴看明堂 → 点穴辨窝钳乳突 → 立向收水
```

另有：「登山看水口，入穴看明堂」「未看山时先看水」「一高一寸山，低一寸水」。

## 2. 五诀总表（龙砂穴水向）

| 诀 | 传统义 | 吉的标准 | 凶的标准 | GIS / 算法对象 |
|----|--------|----------|----------|----------------|
| **龙** | 山脉气脉载体 | 起伏有力、剥换生嫩、过峡有护、入首有情 | 硬直、童断石过独、死龙无起伏 | 山脊线、Strahler 级、过峡极小值 |
| **砂** | 护卫、关水、证穴 | 环抱有情、尖圆方正、官鬼禽曜齐 | 反背、逼压、尖射、破碎 | 扇区峰值、环抱角、水口关锁 |
| **穴** | 气止聚之点 | 形止气蓄、四势端明、乘止 | 五不葬、形凶势逆 | TPI/凹性、窝钳乳突、禁水 |
| **水** | 气之外见、财之象 | 得水有情（抱、聚、曲、关） | 水煞（冲、射、割、反、漏） | 距水、曲率、朝向、水口 |
| **向** | 坐向 / 纳气朝向 | 背山面水、明堂在前 | 向冲煞、向破水 | facing、坐向、面水角 |

**君臣关系（经典表述）**：龙为君，砂为臣；君必位上，臣必伏下。无砂则龙失应，无龙则砂无主。

## 3. 寻龙流程（从祖到入首）

```text
太祖山（发源、势大）
   ↓ 起伏 / 摆动 / 开帐
少祖山（近程主峰）
   ↓ 剥换（粗→细、老→嫩）
父母山（贴穴靠山 ≈ 玄武层级）
   ↓ 胎息孕育
入首（直/横/回/飞/游 等格）
   ↓ 沙交水会
穴星 + 点穴
```

| 环节 | 要点 | 算法提示 |
|------|------|----------|
| 太祖 | 认宗观势，非必登最高再量全局 | 大范围分水岭节点 |
| 少祖 | 二、三节内高峰 | 主脊线上次级隆起 |
| 父母 | **近穴直接靠山** | 50–500 m 后靠峰 |
| 剥换 | 见真龙 | 脊线窗口方差、星体变化 |
| 过峡 | 蜂腰鹤膝，忌风吹水劫 | 脊线鞍部 + 两侧护砂 |
| 入首 | 近穴 2–5 节最紧要 | 末端脊 + 局部坡向 |
| 结穴 | 山水交会、阴阳融凝 | 穴形 + 四象 + 得水 − 水煞 |

**五不葬**（葬经脉络）：童山、断山、石山、过山、独山——对应无生气、气断、气不由土、势不止、无砂水交会。

## 4. 点穴：形与证

### 4.1 穴形四格（窝钳乳突）

| 形 | 几何 | 证佐 |
|----|------|------|
| 窝 | 圆净凹如仰掌 | 有小明堂、晕 |
| 钳 | 两山夹嘴 / U 或 V | 开口宜朝明堂 |
| 乳 | 脊端乳突 | 三面环、余气毡唇 |
| 突 | 圆突小丘 | 边界清、水界分明 |

### 4.2 证穴要素（常并称）

| 证 | 位置 | 作用 |
|----|------|------|
| 乐山 / 鬼星 | 穴后 | 衬托、横龙必有鬼 |
| 官星 | 案/朝后余脉 | 贵证，宜回顾 |
| 禽星 | 水口石 | 关水、名誉 |
| 曜星 | 龙虎尖利余气 | **贵气**非一律作煞（与刑杀射堂要分） |
| 毡唇 / 余气 | 穴前贴身 | 余气铺展 |
| 太极晕 | 穴场微晕 | 土气凝处 |

口诀方向：「无官不贵，无鬼不富，无禽不积，无曜不荣」（证佐齐则地力足）。

## 5. 明堂与水口（点穴两眼）

### 5.1 明堂分层

| 层级 | 范围 | 要求 |
|------|------|------|
| 小明堂 | 穴前最近坦夷 | 圆晕下，辨真假 |
| 内明堂 | 龙虎内 | 元辰水宜曲折不直泻 |
| 中明堂 | 案内 / 龙虎外 | 平正可聚 |
| 外/大明堂 | 案外至朝、水聚 | 忌大旷或逼窄 |

吉：平正、屈曲回环、略凹不积污；凶：破缺、石堆堵塞、内宽外窄失度。

### 5.2 水口

| 概念 | 宜 | 忌 |
|------|----|----|
| 天门（来水） | 开、源远、不见源头 | 急射直灌穴场 |
| 地户（去水） | 闭、重重关锁、之玄 | 直泻、扫帚散乱 |
| 关锁砂 | 华表、捍门、罗星、北辰、狮象龟蛇 | 砂飞水走 |

口诀：**入山寻水口，登穴看明堂**；**天门开，地户闭**。

## 6. 水：得水 vs 水煞（摘要，详文另册）

> 详见 `research/05_sand_water/01_得水与水煞分离模型.md`

| 维度 | **得水（吉项，宜加分）** | **水煞（凶项，宜减分）** |
|------|---------------------------|---------------------------|
| 定义 | 界水止气、有情融注 | 无情冲射割脚反背 |
| 距离 | 适中环抱，不贴脚 | 过近割脚；或冲射无距 |
| 形态 | 玉带、聚堂、之玄、暗拱 | 反弓、直冲、射胁、斜飞、漏槽 |
| 与穴关系 | 在明堂前/侧有情 | 破天心、穿堂、逼基 |
| 算法 | `get_water_score` | `sha_water_penalty`（**独立通道**） |

**不可**把「距水近」同时当成得水满分与水煞满分——二者维度不同，应**分项计分再合成**。

## 7. 四象（摘要，详文另册）

> 详见 `research/02_four_beasts/02_四象体系深入_定位与吉凶.md`

葬经四势：

> 左青龙、右白虎、前朱雀、后玄武。  
> **玄武垂头，朱雀翔舞，青龙蜿蜒，白虎驯俯。**  
> 形势反此，法当破死。

定位以**人面朝（向）**为参照：后靠、前明、左龙、右虎——**非固定东青西白**（固定绝对方位是天文四象，峦头四兽随向转）。

## 8. 立向（向诀）

| 原则 | 说明 |
|------|------|
| 背山面水 | 坐靠父母/玄武，面向明堂与有情水 |
| 收水 | 向宜收吉水、避冲射 |
| 阳宅常用 | 坐北朝南为经典，但以峦头为准可偏东南等 |
| 阴宅 | 更重龙真穴的、向与水口理气（本引擎 v1 以峦头为主） |

算法：优先 `face_water`；次选背靠高地；用户可覆盖 facing。

## 9. 推荐工程流水线（与引擎对齐）

```text
1. DEM + 水系
2. 龙：脊线 / 入首候选
3. 穴候选：干地 + 形态（窝钳乳突代理）
4. 得水分 + 水煞罚（分离）
5. 四象点位与形态分（随 facing）
6. 明堂开阔 + 水口关锁（可选增强）
7. 加权融合 → 平滑场 → 峰值 = 穴
8. 可视化：热力 = 场；紫灰点 = 水煞影响带（非得水带）
```
