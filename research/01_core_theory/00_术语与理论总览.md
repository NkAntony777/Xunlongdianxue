# 01 - 寻龙点穴核心理论与术语体系

> 一份"现代工程化"术语表：把传统风水名词翻译成可计算的 GIS / 几何对象。

## 1. 寻龙点穴的现代定义

- 寻龙点穴，是风水学（旧称堪舆、地理、峦头）术语。古语云："三年寻龙，十年点穴"。其核心目标，是在地表上识别一处"生气凝聚、藏风聚气"的位置——即龙脉结穴。
- 本质上是把"山川形态学"翻译为可计算的"地形格局"：
  - **寻龙** ≈ 沿地形最大起伏方向追踪主脊线，识别太祖山 → 少祖山 → 父母山 → 入首；
  - **点穴** ≈ 在龙脉终止点附近寻找符合藏风聚气条件的局部地形，常见为内凹地形 + 左右护卫 + 前方明堂开阔 + 玉带水环抱。

## 2. "龙砂穴水向"五诀

这是形势派（峦头派）最核心的五要素，源自唐代杨筠松（杨公）：

| 要素 | 传统含义 | 现代可计算对象 |
|------|---------|--------------|
| **龙** | 山脉走向与起伏，是"地气运行的载体" | DEM 上的山脊线（Ridge line）、主脉分叉、剥换关系 |
| **砂** | 穴四周护卫的小山、岗阜、建筑 | 候选穴四周 360°邻域内的局部高程高点 |
| **穴** | 龙气止聚之点，"山水相交、阴阳融凝" | 局部低洼或平缓区域 + 周围山势环抱 + 边界水系 |
| **水** | 河流、沟渠、池塘、湖，"水界龙止，水送龙行" | 流向分析、汇流累积、流路、河曲 |
| **向** | 穴的坐向（朝向），决定纳气方向 | 候选穴的坡向（Aspect）+ 最佳朝向（朝南、朝东南） |

> **核心口诀**："未看山时先看水"，"气乘风则散，界水则止"。

## 3. 祖山体系（可计算分级）

龙脉按等级划分（可算法化为"山脊线层级树"）：

| 等级 | 传统称谓 | 现代对应 |
|------|---------|---------|
| 一级 | 太祖山（发源地） | DEM 全局最高点或最大分水岭节点 |
| 二级 | 少祖山 | 沿主脊线下降过程中第二个显著隆起 |
| 三级 | 父母山 | 距离候选穴最近的主山 |
| 四级 | 入首 | 龙脉最后凸起处，**直接决定穴的坐向** |
| 结穴 | 穴星 | 父母山前、周围有砂环抱的局部微地形 |

- **算法对应**：可通过 watershed → stream order 思路，把山脊线分级：
  - 1 级分水岭 ≈ 太祖山；
  - 沿脊线追踪，遇到"过峡"（两个山顶之间的窄腰）即降低一级；
  - 在最末一级脊线终止点附近搜索"结穴"。

## 4. 龙的"剥换、脱卸、过峡、入首"

四个关键动态过程，对应地形分析中的"分形 + 转折点检测"：

| 过程 | 描述 | 可算法化方式 |
|------|------|-------------|
| **剥换** | 山形由粗变细、石转土、老变嫩 | 沿脊线移动窗口，计算高程方差、坡度变化率、植被覆盖 |
| **过峡** | 两山之间的低凹处（"蜂腰鹤膝"） | 山脊线局部极小值点 + 邻域山头距离比 |
| **开帐** | 祖山展开如鸟翼，脉从中心穿出 | 脊线两侧对称性检测 + 多分支叉点识别 |
| **入首** | 龙脉结穴前最后凸起 | 末端脊线节点 + 局部坡向突变 |

## 5. 穴的形态（"窝、钳、乳、突"四形）

| 形态 | 几何特征 | 算法描述 |
|------|---------|---------|
| **窝穴** | 圆净凹窝，如仰掌 | 局部高程极小值 + 周围 3-5 像素高程梯度收敛 |
| **钳穴** | 两山夹一嘴 | 局部高程 U 形剖面 + 中线两侧有突起 |
| **乳穴** | 山体伸出如乳房 | 沿主脊线末端的凸起（局部高程极大值） |
| **突穴** | 圆突小山包 | 局部高程极大值 + 邻域坡度均匀 + 边界水系清晰 |

## 6. "藏风聚气"的几何本质

- 藏风：四周有屏障阻挡气流（向心凸形/碟形地形）。
- 聚气：气流在水体或地形"止"的位置形成驻点。

> **算法核心**：藏风 ≈ Concave hull / 凹凸性指数；聚气 ≈ 水文分析中的汇流累积最大值点。

## 7. 阴阳基础（《老子》"万物负阴抱阳"）

- 山为阴，水为阳；山南为阳，山北为阴。
- **阳基**（阳宅）：坐北朝南，玄武靠山，朱雀开阔，左右对称。
- **阴基**（阴宅）：同阳基，但更注重"龙真穴的"、朝山秀丽、来龙长远。

> 现代算法：候选穴的"前低后高"模式 + 坡向主朝南/东南 + 左右高程对称性 + 前方 30°-90° 视野开阔。

## 8. 三才说与现代映射

- 天（气、磁场） → 风水中的"生气流注"（目前无直接算法，可由 DTM 派生电磁、辐射等代理变量）。
- 地（龙、砂、水） → 地形分析主要研究对象。
- 人（穴位） → 选址 + 与外部环境的耦合点。

## 9. 流派差异

| 流派 | 重心 | 算法友好度 |
|------|------|-----------|
| 形势派（峦头派） | 山川形态观察 | ★★★★★ 高度可算法化 |
| 理气派（玄空、三合） | 罗盘定向、数理推演 | ★★☆☆☆ 需引入时间维度（年命、九运） |
| 八宅派 | 八个方位与命卦 | ★★★☆☆ 与现代 GIS 兼容 |

> **本引擎定位**：以**形势派（峦头派）**为主，理气派作为可选模块（v2.0 扩展）。

| 口诀 | 程序化含义 |
|------|-----------|
| "三年寻龙，十年点穴" | 寻龙算法可粗，点穴算法必须精 |
| "未看山时先看水" | 先做水文分析（fill → flow dir → flow acc） |
| "气乘风则散，界水则止" | 候选点应在水系汇流点附近 + 周围有屏障 |
| "宁可青龙高万丈，不可白虎抬头望" | 左（东）侧高程可高、右（西）侧不可高 |

| "山环水抱必有气" | 候选点须位于闭合水域+山体围合中 |
| "穴前诸水聚于明堂" | 前方 1-3 km 内有曲水环抱 |
| "大势如蛇行" | 主脊线蜿蜒率要大于阈值（≥1.3） |
