# 四象（四兽）体系深入：定位、吉形、与算法

> 调研日期：2026-07-14（AnySearch + 《葬经》四势 + 本仓库既有量化）  
> 性质：传统文化与可计算化对照，非决策建议。

---

## 1. 两套「四象」不可混

| 体系 | 参照 | 青龙 | 白虎 | 朱雀 | 玄武 |
|------|------|------|------|------|------|
| **天文四象** | 绝对方位 / 二十八宿 | 东 | 西 | 南 | 北 |
| **峦头四兽** | **人面朝（穴之向）** | 左 | 右 | 前 | 后 |

**点穴、砂法用的是峦头四兽**：随 **坐向/朝向** 旋转。  
仅当「坐北朝南」时，左东右西、前南后北，两套重合。

中宫 = 穴（阴宅葬口 / 阳宅核心），四兽守护中宫；**中宫不正则四兽无意义**。

## 2. 葬经四势（元典）

> 夫葬以左为青龙、右为白虎、前为朱雀、后为玄武。  
> **玄武垂头，朱雀翔舞，青龙蜿蜒，白虎驯俯。**  
> 形势反此，法当破死。  
> 故虎蹲谓之衔尸，龙踞谓之嫉主，玄武不垂者拒尸，朱雀不舞者腾去。

| 兽 | 吉形关键词 | 反局（凶） | 几何语义 |
|----|------------|------------|----------|
| **玄武** | 垂头 | 不垂、摆头、拒尸 | 后靠向穴**缓降**，贴靠有情，忌昂然反背 |
| **朱雀** | 翔舞 | 不舞、腾去 | 前场**开敞生动**，案朝有情，忌逼塞或空旷无收 |
| **青龙** | 蜿蜒 | 龙踞嫉主 | 左侧**有起伏、环抱**，忌过高欺主或僵直 |
| **白虎** | 驯俯 | 虎蹲衔尸、抬头 | 右侧**低于青龙、驯服**，忌高昂逼压 |

口诀补充（广泛流传）：

- **宁可青龙高万丈，不可白虎抬头望**  
- 青龙要动，白虎宜静  
- 朱雀不可欺主（前峰不过靠山）  
- 青龙不宜高于玄武主山（臣不过君）

## 3. 各兽深度：层级、距离、作用

### 3.1 玄武（后 · 靠山 · 父母级）

| 项目 | 内容 |
|------|------|
| 角色 | 靠山、稳定、贵人；近场能量锚点 |
| 与祖山链 | 太祖/少祖为远势；**玄武 ≈ 父母山 / 入首后靠**，不是远祖本身 |
| 距离 | 主说 **50–500 m**；多说 **200–500 m** 近而不逼；过近逼压、过远脱气 |
| 高差 | 宜高于穴约数十米量级（常见叙述 +30～100 m 级，随地貌缩放） |
| 坡 | 宜缓（「垂头」）；忌悬崖反弓后靠 |
| 形态 | 圆正、层台；忌破碎、摆头 |
| 乐/鬼 | 横龙结穴重**鬼星**（穴后撑托）；直落有时不要鬼 |
| 算法 | 坐向扇区峰值；dist 窗；Δh 甜区；禁水；`elev_mode=sweet` |

### 3.2 朱雀（前 · 明堂 / 案朝）

| 项目 | 内容 |
|------|------|
| 角色 | 明堂、前案、朝山、前景；可含水（「以水为朱雀」） |
| 分层 | 内明堂（龙虎内）→ 案（近低）→ 朝（远远） |
| 案 | 近而小，齐眉量级叙事，秀而不逼 |
| 朝 | 远而秀，拜伏有情；忌粗恶反背 |
| 水 | 前水宜抱/聚；忌冲心射胁 |
| 算法 | 前向扇区：低坡开阔分 + 案峰高度比靠山 + 得水形态 |

### 3.3 青龙（左 · 龙砂）

| 项目 | 内容 |
|------|------|
| 角色 | 阳、文、伸展；护卫左侧 |
| 吉 | 蜿蜒、环抱过穴、可高于白虎 |
| 凶 | 过高夺气、逼穴、反背、尖射堂 |
| 距离 | 约 0.5–3 穴距量级；实现常取 **80–1200 m** 主搜 |
| 算法 | 左扇区峰值；奖励起伏；硬约束不超过玄武过多 |

### 3.4 白虎（右 · 虎砂）

| 项目 | 内容 |
|------|------|
| 角色 | 阴、收敛；**宜低不宜昂** |
| 吉 | 驯俯、圆顺、略低于青龙 |
| 凶 | **抬头**（≥ 青龙）、蹲逼、尖射 |
| 高度比 | 建议 **白虎/青龙 ≤ 0.85**（高差或绝对，需统一基准） |
| 距离 | 宜近砂；忌 1.5 km+ 乱抓远峰当虎 |
| 算法 | **先青龙后白虎**；`max_elev` 受青龙约束；`elev_mode=lower` |

## 4. 少祖与四象的关系

| 点 | 是否四兽之一 | 说明 |
|----|--------------|------|
| 少祖 | 通常**不是**四兽同位 | 坐向后方更远祖山，证龙来历 |
| 父母/玄武 | 是后靠 | 贴穴 |
| 太祖 | 远势 | 图幅内可缺省 |

主轴可视化常见：**少祖 → 玄武 → 穴 → 朱雀**，旁开龙虎。

## 5. 砂的扩展（与四象一体）

四兽是砂的「四正位」概括；完整砂法还包括：

| 砂类 | 位 | 要点 |
|------|-----|------|
| 案 / 朝 | 前 | 见四象朱雀 |
| 水口砂 | 去水 | 关锁、华表、捍门、罗星、北辰 |
| 罗城 | 周 | 城郭缠护 |
| 侍/送/迎 | 随龙 | 护脉 |
| 官鬼禽曜 | 证穴 | 见总论 |

水口砂：**关主富、锁主贵**；多重关锁更佳。

## 6. 阳宅 vs 阴宅（四兽用法）

| | 阴宅（葬） | 阳宅（居） |
|--|------------|------------|
| 中宫 | 葬口 / 金井 | 宅中心或主气口 |
| 四兽 | 真山真水 | 可建筑、路、桥为砂水 |
| 朱雀 | 案朝明堂 | 前空地、门前景、路 |
| 尺度 | 常更远观龙 | 邻里尺度更重 |
| 本引擎 | 地形四兽为主 | 同一套几何，语义可映射 |

路在形峦中常作**虚水**：环抱路≈玉带，反弓路≈反弓水。

## 7. 算法规格（引擎落地）

### 7.1 输入

- 穴 `(row,col)` 或 `(x,y)`  
- `facing`（朝向，北=0）  
- DEM、可选 water_ban  

### 7.2 方向

```text
sit      = facing + 180   # 玄武
front    = facing         # 朱雀
left     = facing + 270   # 青龙
right    = facing + 90    # 白虎
```

### 7.3 选择顺序（推荐）

```text
1. 穴（评分场峰值，已禁水）
2. facing（面水 / 背山）
3. 玄武（50–500 m，Δh 甜区，禁水）
4. 青龙
5. 白虎（≤0.85 青龙，距离紧）
6. 朱雀（案/朝，宜低于靠山）
7. 少祖（更远、宜高于玄武）
```

### 7.4 与评分的关系

| 模块 | 作用 |
|------|------|
| `score_four_beasts`（扇区统计） | 栅格/候选上的**形态分** |
| `detect_four_beasts`（点位） | **可视化与可解释**的四兽坐标 |
| 综合分 | 四象分 × 权重 + 得水 − 水煞 + 形态… |

点位失败（扇区无峰）≠ 该方向分必为 0，但应降权并在 meta 标明。

### 7.5 验收（阆中类河曲 AOI）

- [ ] 穴不在水上  
- [ ] 玄武在坐向后、距离父母量级  
- [ ] 白虎不高于青龙  
- [ ] 朱雀在前向，不远飞无据  
- [ ] 主轴与 facing 一致

## 8. 与错误实现对照

| 错误 | 后果 | 纠正 |
|------|------|------|
| 用绝对东青西白 | 朝向一偏全错 | 随 facing |
| 玄武抓图幅角点远峰 | 脱气 | dist 硬窗 + 禁边 |
| 白虎 elev 越高越好 | 抬头 | lower + 相对青龙 |
| 穴用候选第一、热力用另一峰值 | 场评与四象脱节 | 统一 score peak |
| 四兽中心在河 | 全局失真 | 禁水后再点四兽 |
